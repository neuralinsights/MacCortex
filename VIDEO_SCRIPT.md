## 视觉风格指南

### 配色方案（与 macOS 一致）

- **背景**: macOS 系统暗色主题（#1E1E1E）
- **主色调**: 蓝色（#007AFF，macOS Accent Color）
- **文字**: 白色（#FFFFFF）/ 浅灰（#E5E5E5）
- **强调色**: 绿色（#34C759，成功状态）、橙色（#FF9500，警告）

### 字体

- **标题**: SF Pro Display（macOS 系统字体，Bold）
- **正文**: SF Pro Text（Regular）
- **代码**: SF Mono（等宽字体）

### 动画效果

- **淡入/淡出**: 300ms ease-in-out
- **打字效果**: 50-200 ms/字符
- **按钮点击**: 缩放动画（scale 0.95 → 1.0，100ms）
- **加载动画**: 旋转进度条（360° 循环，1s）

## 音频/音乐建议

### 背景音乐

- **风格**: 轻快、现代、科技感
- **BPM**: 120-140（节奏明快）
- **音量**: -20dB（不掩盖旁白）
- **推荐曲目**:
  - "Tech Inspire" (Epidemic Sound)
  - "Digital Flow" (Artlist)
  - 或使用无版权 lofi 音乐

### 旁白（可选）

- **语速**: 中速（约 200 字/分钟）
- **语调**: 友好、专业、自信
- **录音质量**: 48kHz, 16-bit WAV
- **总字数**: < 30 字（15 秒时长）

## 录制设备要求

### 硬件

- **Mac 电脑**: MacBook Pro/Air M1+ (Apple Silicon 优化)
- **屏幕**: 2560x1440 或更高（录制 1920x1080 区域）
- **麦克风**（如需旁白）: Blue Yeti / Rode NT-USB

### 软件

- **屏幕录制**: QuickTime Player（系统内置）/ ScreenFlow（专业版）
- **视频编辑**: Final Cut Pro / iMovie / DaVinci Resolve
- **字幕工具**: Arctime / Subtitle Edit

## 录制步骤（快速指南）

### 第 1 步：准备环境

```bash
# 1. 清理桌面（移除杂乱图标）
# 2. 启用 macOS 暗色模式
系统设置 → 外观 → 深色

# 3. 关闭通知（勿扰模式）
控制中心 → 勿扰模式 → 开启

# 4. 启动 MacCortex Backend
cd /Applications/MacCortex.app/Contents/Resources/Backend
python3 src/main.py

# 5. 预热模型（首次推理）
curl -X POST http://localhost:8000/execute -d '{"pattern_id":"summarize","text":"test",...}'
```

---

### 第 2 步：录制屏幕

```bash
# 使用 QuickTime Player
1. 打开 QuickTime Player
2. 文件 → 新建屏幕录制
3. 选择录制区域（1920x1080 居中）
4. 点击录制
5. 按照分镜脚本执行操作（15 秒）
6. Command+Control+Esc 停止录制
```

---

### 第 3 步：后期制作

1. **导入素材**: 将录制视频导入 Final Cut Pro
2. **剪辑时间线**: 精确裁剪至 15 秒
3. **添加文字叠加**: 使用分镜脚本中的文字（中英双语）
4. **添加背景音乐**: 音量 -20dB
5. **添加旁白**（可选）: 录制后混音
6. **颜色校正**: 确保色彩一致（macOS 暗色主题）
7. **导出视频**:
   - 格式: H.264
   - 分辨率: 1920x1080
   - 帧率: 60 fps
   - 码率: 10 Mbps

---

### 第 4 步：发布与分发

**发布渠道**:
- ✅ **GitHub README**: 嵌入 GIF（转换为 GIF 后）
- ✅ **官网首页**: 直接播放 MP4（自动循环）
- ✅ **社交媒体**: YouTube Short / X(Twitter) / 微博

**GIF 转换**（用于 GitHub）:
```bash
# 使用 ffmpeg 转换为 GIF（< 10 MB）
ffmpeg -i maccortex_demo.mp4 -vf "fps=30,scale=1280:-1:flags=lanczos" \
  -loop 0 maccortex_demo.gif

# 优化 GIF 大小
gifsicle -O3 --lossy=80 maccortex_demo.gif -o maccortex_demo_optimized.gif
```

### 故障排查

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 录制画面模糊 | 分辨率过低 | 确保录制 1920x1080 或更高 |
| 应用启动慢 | 首次冷启动 | 预先启动 MacCortex 并预热模型 |
| 推理超时 | 网络问题/模型未加载 | 检查 Backend 日志，重启 Backend |
| GIF 体积过大 | 帧率过高/未优化 | 降低帧率到 30 fps，使用 gifsicle 优化 |
